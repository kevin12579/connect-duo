React MainPage 컴포넌트를 만들어줘.

[프로젝트 구조]
- src/pages/Main/MainPage.jsx
- src/pages/Main/MainPage.css

[레이아웃 요구사항]
전체는 세로 구조 (flex column)

1️⃣ 상단 (AI 챗봇 영역)
- 회색 배경 카드 형태
- 왼쪽: 로고 또는 챗봇 아이콘
- 가운데: "무엇을 도와드릴까요?" 텍스트
- 아래: 검색 입력창 (input + 돋보기 아이콘)
- 둥근 카드 스타일 (border-radius 20px)

2️⃣ 중단 (카테고리 메뉴)
- 가로 4개 버튼
- 순서: 로그인 / 프로필 / 랭킹 / 상담
- 원형 버튼 디자인
- 아이콘 + 텍스트 세로 정렬
- 클릭 시 선택된 버튼 강조 (배경색 darker, scale 1.05)
- hover 효과 추가

3️⃣ 하단 (콘텐츠 영역)
- 큰 카드 박스
- 선택된 메뉴에 따라 다른 컴포넌트 표시
- 페이지 이동(router) 하지 말고 state로 화면 전환

[기능 요구사항]
- useState로 activeTab 관리
- 버튼 클릭 시 activeTab 변경
- 조건부 렌더링:
  login → <LoginPage />
  profile → <ProfilePage />
  ranking → <RankingPage />
  consult → <ConsultPage />

[스타일]
- flexbox 사용
- 반응형
- 카드 UI 느낌
- 부드러운 그림자 box-shadow
- 전체 여백 padding 24px
- 색상: #f5f5f5, #e0e0e0, #333 계열
- 깔끔한 modern dashboard 디자인

[추가]
- MainPage.css 분리
- 더미 컴포넌트 4개도 함께 생성 (각각 "로그인 화면" 같은 텍스트 표시)
